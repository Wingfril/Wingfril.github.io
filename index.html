    BeeAProphet

    Using a Python backend and Django, this site allows the user to predict changes in bee population based on temperature. Bees pollinate more then $15 billion a year to the economy by pollinating important crops [1], but they are vanishing mysteriously due to Colony Collapse Disorder.
